```markdown
【问题描述】

NFA→DFA问题的一种描述是：编写一个程序，输入一个非确定有穷自动机（NFA），输出与NFA等价的确定有穷自动机（DFA）。

【基本要求】设置FA初始状态X，终态Y，过程态用数字表示：0 1 2 3………;如有多个终态，则以此命名为Y，Y1，Y2，......

【输入形式】输入一个非确定有穷自动机（NFA）

【输出形式】输出与NFA等价的确定有穷自动机（DFA）

【样例输入】
X X-~->3
Y
0 0-a->1
1 1-b->2
2 2-b->Y
3 3-~->0 3-a->3 3-b->3

【样例输出】
X X-a->0 X-b->1
Y Y-a->0 Y-b->1
0 0-a->0 0-b->2
1 1-a->0 1-b->1
2 2-a->0 2-b->Y


【样例说明】符号”~“表示空串
```
