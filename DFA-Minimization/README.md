```markdown
【问题描述】

DFA化简问题的一种描述是：编写一个程序，输入一个确定的有穷自动机（DFA），输出与DFA等价的最简的确定有穷自动机（DFA）。

【基本要求】设置DFA初始状态X，终态Y，过程态用数字表示：0 1 2 3………

【样例输入】
X X-a->0 X-b->1
Y Y-a->0 Y-b->1
0 0-a->0 0-b->2
1 1-a->0 1-b->1
2 2-a->0 2-b->Y

【样例输出】
X X-a->0 X-b->X
Y Y-a->0 Y-b->X
0 0-a->0 0-b->2
2 2-a->0 2-b->Y




【样例说明】符号”~“表示空串
```
